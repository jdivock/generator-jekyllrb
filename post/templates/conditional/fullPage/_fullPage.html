Huge.FullPageWidget

'Allo, I'm a full page widget

Prereqs

  • Transit

Compatibility

  • Chrome, Opera, lynx

Usage

{% highlight js %} $(document).ready(function() { new FullPageWidget(); }); {% endhighlight %} {% highlight sass %} section { position: relative; } html, body, article { height: 100%; } html, body { overflow: hidden; } {% endhighlight %}

Options

Name type default description
